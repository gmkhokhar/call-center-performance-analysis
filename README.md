# 📊 Call Center Performance Analysis

## 🎯 Objective
Analyze call center data to improve operational efficiency, customer satisfaction, and agent performance by identifying key trends and performance gaps.

---

## 🛠 Tools Used
- Microsoft Excel  
- Power Query  
- Pivot Tables  
- Data Visualization (Dashboard)  

---

## 📁 Dataset Overview
The dataset contains 5,000 records of call center operations, including:

- Call_ID → Unique identifier for each call  
- Date → Timestamp of the call  
- Agent → Call handling agent  
- Department → Department assigned  
- Answered → Whether call was answered (1/0)  
- Resolved → Whether issue was resolved (1/0)  
- Speed_of_Answer → Time taken to answer (seconds)  
- Talk_Duration_Seconds → Call duration  
- Satisfaction_Rating → Customer satisfaction score  

---

## 🧹 Data Cleaning & Preparation
Performed using Power Query:

- Standardized column names  
- Converted data types  
- Handled missing values  
- Created KPI columns:
  - Answer Rate  
  - Resolution Rate  
  - Resolution Efficiency  
- Extracted time-based features (Hour, Day, Month)  
- Converted call duration into seconds for analysis  

---

## 📈 Data Analysis
Using Pivot Tables, the following insights were generated:

- Agent performance comparison  
- Department-level efficiency  
- Impact of response time on satisfaction  
- Call volume trends by hour  
- Relationship between talk duration and resolution  

---

## 📊 Dashboard
An interactive Excel dashboard was created featuring:

- KPI cards (Answer Rate, Resolution Rate, Satisfaction Score)  
- Agent & Department performance charts  
- Response time vs satisfaction analysis  
- Call volume trends  
- Interactive slicers (Agent, Department, Day)  

---

## 🔍 Key Insights

- Faster response times significantly improve customer satisfaction  
- Performance varies widely between agents  
- Certain departments underperform in resolution and satisfaction  
- Peak hours increase workload and reduce efficiency  
- Moderate call duration leads to better resolution rates  

---

## 💡 Recommendations

- Train underperforming agents using top performers as benchmarks  
- Improve response time through process optimization  
- Allocate more staff during peak hours  
- Standardize call handling practices
